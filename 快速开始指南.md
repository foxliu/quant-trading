# 快速开始指南

本指南帮助你在5分钟内运行量化交易系统。

## 前置条件

- Go 1.24+ 已安装
- 基本的Go语言知识

## 步骤1: 编译项目

```bash
cd quant-trading
go build ./...
```

预期输出: 无错误信息

## 步骤2: 运行演示程序

```bash
go run cmd/demo/main.go
```

预期输出:
```
账户信息: ID=demo_account, 现金=100000.00, 权益=100000.00
策略引擎演示完成
```

## 步骤3: 运行回测

```bash
# 编译回测程序
go build -o bin/backtest cmd/backtest/main.go

# 运行回测
```

创建文件 `internal/infrastructure/strategy/examples/my_strategy.go`:

```go

    "quant-trading/internal/domain/market"
    "quant-trading/internal/domain/strategy"
)

type MyStrategy struct {
    name string
}

func NewMyStrategy(name string) *MyStrategy {
    return &MyStrategy{name: name}
}

func (s *MyStrategy) Name() string {
    return s.name
}

func (s *MyStrategy) OnInit(ctx strategy.Context) error {
    return nil
}

func (s *MyStrategy) OnMarketEvent(ctx strategy.Context, evt market.Event) ([]strategy.Signal, error) {
    // 你的策略逻辑
    return nil, nil
}

func (s *MyStrategy) OnStop(ctx strategy.Context) error {
    return nil
}
```

## 常见问题

### Q: 编译失败怎么办?

A: 确保Go版本 >= 1.24,并检查GOPATH设置。

### Q: 回测没有产生交易怎么办?

A: 这是正常的,示例数据可能不满足策略条件。你可以修改策略参数或数据生成逻辑。

### Q: 如何添加真实数据?

A: 修改`cmd/backtest/main.go`中的`generateTestData`函数,从CSV或数据库加载真实行情数据。
